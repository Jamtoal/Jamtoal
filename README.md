### Coding Time Now

```Julia
输入：一个二进制向量 x = (x1, x2, ..., xn)  // 输入信号
权重：一个实数向量 w = (w1, w2, ..., wn)    // 输入权重
阈值：一个实数 t                            // 阈值

计算加权和：
s = w1*x1 + w2*x2 + ... + wn*xn

应用阈值函数进行激活：
如果 s >= t，则输出为 1
否则，输出为 0

```
